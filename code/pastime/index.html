Pastime

Live-updating timespans using unobtrusive JavaScript.

These will update automatically

Examples



Thu Oct 20 23:30:04 -0700 1700
10/16/2010 1:15

Usage

<span class="foo">Thu Oct 20 23:30:04 -0700 1700</span>
$('.foo').pastime({
  interval: 1,  // seconds
  includeSeconds: true,  // show seconds in output
  includeAgo: true  // append ago to output
});
